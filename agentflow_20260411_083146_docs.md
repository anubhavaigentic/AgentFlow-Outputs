### Logging Configuration

#### `setup_logging(level: str = "INFO") -> logging.Logger`

Configures the root logging system and returns a named logger for the current module.

| Parameter | Type  | Default  | Description                                              |
|-----------|-------|----------|----------------------------------------------------------|
| `level`   | `str` | `"INFO"` | Logging verbosity: `"DEBUG"`, `"INFO"`, `"WARNING"`, `"ERROR"` |

**Returns:** A `logging.Logger` instance bound to `__name__`.

**Behaviour:**
- Uses `logging.basicConfig` to apply a timestamped format to all handlers.
- Timestamp format: `YYYY-MM-DDTHH:MM:SS`.
- Output stream: `sys.stdout`.
- Falls back to `INFO` if an unrecognised level string is supplied.

**Log format:**
```
2024-05-01T12:00:00 | INFO     | __main__ | Your message here
```

A module-level `logger` object is instantiated immediately after this function is defined, making it available throughout the module without additional setup.

#### `load_data(path: Path) -> list[dict[str, Any]]`

Reads a JSON file from disk and returns its contents as a list of dictionaries.

| Parameter | Type   | Description                    |
|-----------|--------|--------------------------------|
| `path`    | `Path` | Path to the input JSON file.   |

**Returns:** `list[dict[str, Any]]` — a list of record dictionaries.

**Raises:**

#### `process_records(records, *, filter_key=None, filter_value=None) -> list[dict[str, Any]]`

Processes a list of record dictionaries, optionally filtering them by a key/value pair, and marks each surviving record with a `_processed: true` flag.

| Parameter      | Type                    | Default | Description                                                         |
|----------------|-------------------------|---------|---------------------------------------------------------------------|
| `records`      | `list[dict[str, Any]]`  | —       | The source list of record dictionaries.                             |
| `filter_key`   | `str \| None`           | `None`  | If set, only records where `record[filter_key] == filter_value` are kept. |
| `filter_value` | `Any`                   | `None`  | The required value for `filter_key`. Ignored when `filter_key` is `None`. |

**Returns:** A new list of processed records. Original records are not mutated; each result entry is a shallow copy with `_processed` added.

**Logging:**
- `DEBUG` per skipped record.
- `INFO` summary of retained vs. total record count.

**Processing pipeline per record:**

```
Input record
    │
    ▼
[filter_key set?] ── No ──────────────────────────────────────────────►
    │                                                                   │
   Yes                                                                  │
    │                                                                   │
    ▼                                                                   │
[record[filter_key] == filter_value?]                                  │
    │                                                                   │
   No → skip (DEBUG log)                                               │
    │                                                                   │
   Yes ─────────────────────────────────────────────────────────────►  │
                                                                        ▼
                                                             {**record, "_processed": True}
                                                                        │
                                                                        ▼
                                                                   results list
```

#### `save_data(data: list[dict[str, Any]], path: Path) -> None`

Serialises a list of records to a pretty-printed JSON file.

| Parameter | Type                   | Description                            |
|-----------|------------------------|----------------------------------------|
| `data`    | `list[dict[str, Any]]` | The records to serialise.              |
| `path`    | `Path`                 | Destination file path.                 |

**Behaviour:**
- Automatically creates any missing parent directories (`mkdir(parents=True, exist_ok=True)`).
- Writes with `indent=2` and `ensure_ascii=False` to support Unicode content.
- Raises `OSError` on write failure (caught and handled in `main`).

**Logging:** Emits an `INFO` message with the destination path and record count.
